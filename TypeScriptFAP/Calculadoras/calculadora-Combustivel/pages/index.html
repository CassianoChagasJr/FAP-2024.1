Bem vindo ao Comparador de Combustível este projeto tem como objetivo a construção de uma calculadora que mostre qual combustível vale colocar em seu veículo utilizando as tecnologias HTML, CSS e TypeScript. A escolha dessas linguagens visa consolidar os conhecimentos adquiridos em desenvolvimento web e explorar a tipagem estática proporcionada pelo TypeScript para criar um código mais robusto e seguro. Através deste projeto, busco aprimorar minhas habilidades em manipulação do DOM, criação de interfaces intuitivas e aplicação de lógica de programação para resolver problemas matemáticos básicos.
